Museomix webdoc embed examples

The goal of these examples is to provide different ways to embed Museomix webdoc in your own page

  • Iframe #2 (fullscreen) : uses 99% of available height, looks like we were on MSX website
    The recommanded solution, but it needs to have ability to upload a file on the webserver
  • Iframe #1 : fixed-size iframe, to include it on a thight website design
    Offers the ability to fit with the websites design, but doesn't use all available size, and iframes are sometimes blocked by the websites CMS.
  • Javascript : dynamically creates iframe
    Useful if iframes are blocked, but needs Javascript not to be blocked.
